About

Schema is the offline world editor for the Ambient engine. It provides the full game environment without requiring a server connection, making it the primary tool for world design, content creation, and gameplay testing.

Features

World Editing

Full voxel editing with all block types, orientations, and variations. Place, paint, and sculpt terrain directly.

Saga Testing

Test saga arcs, dialogue trees, and character interactions locally before deploying to a live server.

Block Extensions

Configure and test machines, furnaces, and other interactive block types with the extension editor.

Content Pipeline

Import models, configure textures, and build content packs for any Ambient Games title.
