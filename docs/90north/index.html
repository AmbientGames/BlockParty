About

90° North: Dash & Cache is a multiplayer speed run and geocaching game set in a procedurally generated arctic world. Race across frozen terrain to reach the North Pole as fast as you can, then place geocaches throughout the landscape for other players to discover. Compete for the fastest times and the most creative cache placements.

Built on the Ambient engine with real-time terrain generation, dynamic climate, and community-driven geocache discovery.
